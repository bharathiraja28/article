## Getting Started

<!-- This is an example of how you may give instructions on setting up your project locally.

### Prerequisites

* rails
  ```sh
  gem install rails
  ```
* check version
  ```sh
  rails -v
  ```
  ```sh
  ruby -v
  ```
  
  

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bharathiraja28/article.git
   ```
   
2. Install Bundle
   ```sh
   bundle install
   ```
3. Update Bundle
   ```sh
   bundle update
   ```
4. Migrate Database
   ```sh
   rails db:migrate
   ```

### To Run

   Clone the repo
   ```sh
   rails s
   ```
